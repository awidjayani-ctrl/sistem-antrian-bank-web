# Sistem Antrian Bank Berbasis Web (PHP)

Proyek ini merupakan implementasi sistem antrian digital berbasis web menggunakan **HTML, CSS, dan JavaScript (client-side)** dengan penyimpanan data sementara melalui **LocalStorage**.  
Tujuan utama sistem ini adalah mempermudah proses layanan nasabah di bank melalui simulasi antrian digital tanpa server eksternal.
